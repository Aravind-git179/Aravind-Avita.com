How to set up and use

Installation and Setup Guide for Graphical Figure App.

Hardware and Software Required

The basic hardware required for running the Graphical Figure App includes a dual-core processor (such as an Intel Core i3 or AMD equivalent), at least 4 GB of RAM, and 500 MB of free disk space for the app installation, with an additional 1 GB recommended for MySQL and saved files. The system should operate on Windows 7 or later (64-bit recommended). A monitor with a minimum resolution of 1280x720, along with a standard keyboard and mouse, is essential for interaction. For optimal performance, it is recommended to use a quad-core processor (e.g., Intel Core i5 or higher), 8 GB of RAM, and a Solid State Drive (SSD) with at least 1 GB of free space. A Full HD monitor (1920x1080 resolution) and a stable internet connection are also suggested for a smoother user experience. MySQL installation is mandatory for database management, and a printer may be optionally required for printing balance sheets or reports.

Setup Steps

Step 1: Download MySQL Visit the MySQL website and download the MySQL server, as we do not have any centralized databases.
Step 2: Download the Graphical Figure App Click on the download link for the Graphical Figure app to get the Graphical_Figure_setup_folder_version3.7z file.
Step 3: Extract the ZIP File Open the ZIP file and extract its contents. Inside, find the setup.exe file and open it.
Step 4: Agree to the Terms and Conditions Read the terms and conditions, then agree to them to proceed with the installation.
Step 5: Provide MySQL Credentials Enter your MySQL username and password when prompted. Click Submit to proceed.
Step 6: Choose Installation Location Select the location on your system where you want to store the application files. Click Save to confirm.
Step 7: Create Graphical Figure Account Click Yes to proceed. Create your Graphical Figure ID and password, then click the Create button to complete your account setup.
Step 8: Complete Installation Click Install to begin the installation process. Once installed, click OK to finish and launch the app.
